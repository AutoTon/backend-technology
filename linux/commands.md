# linux 常用命令

## 1. 磁盘相关

### 1.1 查看子目录磁盘使用情况

```
du -h --max-depth=1 <dir>
```
### 1.2 查看各个挂载盘的使用情况

```
df -hl
```

## 3. 网络相关

### 3.1 统计连接数

```
netstat -n | awk '/^tcp/ {++S[$NF]} END {for(a in S) print a, S[a]}'
```

### 3.2 查看指定端口是否被占用

```
lsof -i:<port>
```
> 注意： 该命令只能查看当前用户下的进程是否占用了指定端口，若要查询该端口是否真正占用，需要加上sudo

若提示命令未找到，执行下面命令安装`lsof`

```
sudo yum install lsof
```
